# DevOps Capstone Project

## Overview

This project demonstrates the implementation of DevOps practices using automation, testing, containerization, and Continuous Integration/Continuous Deployment (CI/CD).

## Features

* Automated Testing
* Continuous Integration
* Continuous Deployment
* Docker Containerization
* GitHub Actions Workflow

## Technologies Used

* Python
* Flask
* Docker
* GitHub Actions

## Running the Application

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Run tests:

```bash
nosetests
```
